House Price Prediction

Overview

This project uses a Random Forest model to predict house prices based on room count, land size, and construction year.

Installation

Requires Python, pandas, and sklearn. Install dependencies with:

Copy code
pip install pandas sklearn
Usage

Run predict_price.py and input the number of rooms, land size, and year built for a price prediction.
